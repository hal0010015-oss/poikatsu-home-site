About

ポイ活ホームについて

ポイ活ホームは、ユーザーが登録した外部アプリやWebサービスの日々の確認を忘れにくくするための管理アプリです。今日確認したいアプリをリスト化し、アプリを開いたあとに、ポイ活ホームに戻って「完了」「まだ」「スキップ」を手動で記録できます。本アプリは、外部アプリの自動操作やポイントの自動回収を行うものではありません。ユーザー自身の毎日の確認作業をサポートするためのチェックリストです。

安心してご利用いただくために

ポイ活ホームが「行わないこと」を明確にしています。
✓外部アプリの自動操作は行いません
✓ポイントの自動回収は行いません
✓画面認識は行いません
✓アクセシビリティ権限での外部操作は行いません
✓外部サービスのポイント数・金額・残高は取得しません
✓アプリ内でポイント付与・交換・金銭の授受は行いません

How to use

FAQ

よくある質問

Q.ポイントは自動で回収されますか？
A.いいえ。ポイ活ホームは自動回収アプリではありません。ユーザー自身の確認作業を忘れにくくするためのチェックリスト・リマインダーです。
Q.外部アプリを自動操作しますか？
A.いいえ。外部アプリの自動操作、画面認識、アクセシビリティ権限を使った操作は一切行いません。
Q.ポイント数や金額を取得しますか？
A.いいえ。外部サービスのポイント数、金額、残高、アカウント情報は取得しません。
Q.データはどこに保存されますか？
A.登録したアプリ、チェック状態、履歴、通知時刻などはすべて端末内に保存されます。ウィジェット連携のため、一部データはApp Groups領域に保存されます。独自サーバーへの同期は行いません。
Q.アプリが開かない場合は？
A.登録されたURL Scheme、App Store URL、Web URLの状態により、App StoreやWebページが開く場合があります。アプリ内のリンク修正機能で調整できます。解決しない場合はお問い合わせください。
Q.広告や紹介リンクはありますか？
A.本アプリは「広告」と明記したバナー広告を表示しています（プレミアムをご利用の場合は非表示になります）。PR表記付きの紹介リンクは現在含まれていません。将来追加する場合は、利用を任意とし、本アプリの主要機能の利用条件にはしません。
Q.無料で使えますか？
A.はい。基本機能は無料でご利用いただけます（無料版は登録アプリ5個まで）。買い切り¥500のプレミアムでは、登録数無制限やタスク単位でのチェック、巡回モードなどの機能をご利用いただけます。

Privacy

プライバシーポリシー

最終更新日: 2026年7月19日

1. はじめに

ポイ活ホームは、ユーザーが登録した外部アプリやWebサービスの日々の確認を忘れにくくするためのチェックリスト・リマインダーアプリです。

本アプリは、外部アプリの自動操作、ポイント自動回収、画面認識、アクセシビリティ権限を利用した操作、外部サービスのポイント残高取得を行いません。

2. アプリ内に保存する情報

本アプリは、以下の情報を端末内に保存します。

  • 登録したアプリ名 / カテゴリ
  • App Store URL / Web URL / URL Scheme
  • タスク名
  • 完了、未チェック、スキップなどのチェック状態
  • アプリ別リセット時刻 / 通知時刻
  • 起動検証ログ
  • 連続日数、完了日、履歴

本アプリは、ポイント数、金額、外部サービスの残高、外部アプリ内の操作内容を自動取得しません。

3. 保存場所

アプリのデータは端末内、およびウィジェット連携のためのApp Groups領域に保存されます。本アプリは、現時点でアカウント登録、独自サーバーへの同期、クラウド保存を行いません。

4. お問い合わせ時の情報

本アプリの「お問い合わせ・リクエスト」画面では、ユーザーが任意で内容と返信先を入力できます。ユーザーがメール送信またはコピー操作を行った場合、入力内容は問い合わせ本文として扱われます。送信先は afi811944@gmail.com です。

5. 広告（Google AdMob）

本アプリは、Google AdMobによるバナー広告を表示しています。広告の配信、計測、不正防止のため、Google Mobile Ads SDKによって端末情報、広告識別子、利用状況、診断情報、おおよその位置情報などが扱われる場合があります。パーソナライズされない広告のみを配信しており、法令またはGoogleの方針により同意が必要な地域では、広告表示前にプライバシー選択画面を表示します。プレミアムをご利用の場合、広告は表示されません。詳細はGoogleのポリシーをご確認ください。

Google ポリシーと規約: https://policies.google.com/
Google AdMob: https://admob.google.com/

6. 外部リンク

本アプリから外部アプリ、App Store、Webサイトを開くことがあります。リンク先のサービス内容、ポイント付与、キャンペーン、アカウント状態、情報の取り扱いは各サービス提供者が管理します。リンク先のプライバシーポリシーや利用規約をご確認ください。

7. データ削除

端末内に保存された本アプリのデータは、iOSのアプリ削除により削除できます。今後、アプリ内にデータリセット機能を追加する場合があります。

8. 本ポリシーの変更

本ポリシーは、機能追加、広告SDKの変更、法令対応などにより更新される場合があります。
